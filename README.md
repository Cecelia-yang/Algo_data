# Algo & Data structure II — CHU

This repository contains material for an Algo & Data structure course.

The project is organized by course folders (`course_XX`) and includes:
- exercises,
- teaching notes,
- and slides.

## Course folder structure

```
course_XX/
├── exercice/
│   ├── exercise.py              # Student exercise (starter version)
│   └── exercice_correct.py      # Exercise correction
├── teaching_note/
│   └── teaching_note.md
└── slides/
```
